# odin-recipes

Odin Recipes Project

This is my first project from The Odin Project.
The goal of this project was to build a simple recipe website using basic HTML.


Description

In this project, I created a small website that displays different recipes.
Each recipe page includes ingredients, steps, and images.

This project helped me practice structuring web pages and using Git from the terminal.


Features

- Multiple recipe pages.
- Use of unordered lists for ingredients.
- Use of ordered lists for steps.
- Images included in each recipe.
- Navigation between pages.


Technologies Used

- HTML5.
- Git.
- GitHub.
- Linux Terminal.


What I Learned

- How to structure a basic HTML website.
- How to use lists (ul and ol) properly.
- How to add and display images in HTML.
- How to use Git commands:.
- git add.
- git commit.
- git push.
- How to manage a repository on GitHub from the terminal.
